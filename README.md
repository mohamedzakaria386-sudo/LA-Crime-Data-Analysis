# 🔍 LA Crime Data Analysis

## Overview
Exploratory Data Analysis (EDA) project on a real-world 
Los Angeles crime dataset containing 5,000+ records.

## 🛠️ Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Key Findings
- Peak crime hour is *12 PM (Noon)*
- *77th Street* has the most night crimes (10PM–4AM)
- *26-34 age group* are the most frequent victims
- *Theft of Identity* is the most common crime type
- Crime spikes sharply from *May to June*
- Nearly equal gender split: 51.1% Male, 47.6% Female

## 📁 Dataset
Los Angeles crime records with 12 features including:
Area, Crime Type, Victim Age/Sex/Descent, Weapon, and Time.

## 🔄 Project Steps
1. Data Loading & Exploration
2. Data Cleaning & Preprocessing
3. Feature Engineering
4. Visualization & Insights
